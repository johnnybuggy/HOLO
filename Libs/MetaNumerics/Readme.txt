Meta.Numerics

The Meta.Numerics library brings scientific computing to the .NET platform. It offers an object-oriented API for matrix algebra, advanced functions of real and complex numbers, and statistics.

For more information, visit the http://www.meta-numerics.net website. There you will find detailed specifications, documentation, and myriad web calculators illustrating the capabilities of the Meta.Numerics library.
